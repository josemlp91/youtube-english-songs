 # YouTube English Songs

 Aprende inglés mediante canciones de YouTube con subtítulos interactivos y glosario de vocabulario.

 **URL de la aplicación (GitHub Pages):**
 `https://josemlp91.github.io/youtube-english-songs/`

 ## Desarrollo

 - Ejecutar en Chrome: `make run`
 - Instalar dependencias: `make deps`
 - Analizar: `make analyze`
 - Formatear: `make format`
 - Tests: `make test`

 ## Build y deploy

 - Build local: `make build-web-local`
 - Servir local: `make serve`
 - Build para GitHub Pages: `make build-web`
 - Deploy: `make deploy GITHUB_USER=josemlp91`
